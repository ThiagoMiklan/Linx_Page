# Linx_Page

- 1º Passo - Clonar o repositório Linx Page
- 2º Passo - Abrir o prompt no diretório princpial e instalar todas as depêndencias (npm i)
- 3º Passo - Para iniciar a aplicação, basta executar npm start e esperar o build e abertura da página

# Comportamento

- 1 - Ao clicar em "Ainda mais produtos aqui!" todas as fotos serão atualizadas conforme nextPage
- 2 - Para tratar os inputs de amigo e email ao final da página, foi utilizado React e adicionado uma mensagem
    embaixo do input para avisar se a entrada está:
    - OK (cor verde)
    - Caracter não alfabético informado (amigo) (cor vermelha)
    - Email não está no formato correto (cor vermelha)
    - Máximo de caracteres ultrapassado (cor vermelha)


# Pacotes utilizados

- 1 - React - Utilizado para facilitar na construção da interface gráfica
- 2 - Webpack - Para testar a página
